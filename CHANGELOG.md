# Changelog

All notable changes to Polyglow will be documented in this file.

This project follows the spirit of Keep a Changelog and uses semantic versioning once public releases begin.

## Unreleased

## v1.0.0-beta.1 - 2026-05-14

### Added

- Static multilingual Astro content site with ten locale prefixes.
- Tailwind CSS 4, Astro-native UI components, Lucide icons through Astro Icon, and Pagefind search.
- SEO metadata, RSS, robots, sitemap, JSON-LD, and locale alternates.
- Polyglow-inspired glassmorphism card system, dynamic color extraction, and visual regression coverage.
- Optional remote image and R2 helper scripts.
- Release readiness, content authoring, theme configuration, analytics, Cloudflare, demo content, and roadmap documentation.
- Theme adoption helpers: minimal content template, `theme:reset`, `theme:check`, fork guide, and beta release notes.

### Changed

- Home page now uses a compact three-card first page with pagination.
- Public theme architecture is Astro-first and does not require a client framework, CMS, comments, membership, login, or forced object storage.
