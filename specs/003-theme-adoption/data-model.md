# 数据模型：主题采用体验

## 采用指南

- **字段**：标题、目标用户、前置条件、必须修改文件、推荐修改文件、验证命令、常见问题。
- **关系**：链接到主题配置、内容写作、图片策略、Cloudflare 部署和发布清单。
- **规则**：必须提供 10 分钟内能完成的最短路径。

## 最小内容模板

- **字段**：文章、About 页面、作者资料、图片引用、README 说明。
- **关系**：映射到 `src/content/posts`、`src/content/pages`、`src/content/authors`。
- **规则**：模板复制后能通过内容 schema 和静态构建。

## 重置流程

- **字段**：输入模板目录、目标内容目录、执行模式、输出摘要、失败信息。
- **关系**：读取 `examples/minimal-content/`，影响 `src/content/`。
- **规则**：默认不执行不可恢复删除；失败时输出明确文件路径。

## 检查规则

- **字段**：规则编号、检查对象、默认值模式、严重级别、修复提示。
- **关系**：检查 README、配置、作者、页面、文章和环境示例。
- **规则**：每条规则具备确定性结果和测试覆盖。

## beta 发布包

- **字段**：版本号、tag、变更记录、发布说明、验证记录、已知提示。
- **关系**：引用 CHANGELOG、release preflight、quality 记录和 README。
- **规则**：tag、CHANGELOG 和 Release notes 使用同一版本号。
